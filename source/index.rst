Welcome to Voiser's documentation!
===================================

**Voiser** is an AI-powered service that converts text to audio and converts audio to text. You can access the web page via `voiser.net <https://voiser.net>`_.

.. note::

   To use the Voiser APIs, you must create an account on `voiser.net <https://voiser.net>`_ and purchase a package. Then you can use the contact page for API access.
